# UI-layout for RanAswanu web app

# Important !
## Color Scheme
``#6D6D6D - Light gray
  #058A00 - Green (Primary)
  #404040 - Dark gray
  #FFFFFF - White
``
## Use of variables
`` examples:
        :root {
          --main-bg-color: coral;
        }

        #div1 {
          background-color: var(--main-bg-color);
        }

        #div2 {
          background-color: var(--main-bg-color);
        }
``
